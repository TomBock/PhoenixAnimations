# PhoenixAnimations

### Commands
- /animation - Öffnet das Animationsmenü
- /animation <Name> - Spielt die angegebene Animation ab (sofern der Spieler die Berechtigung hat)

### Permissions
- phoenixanimations.<Name> - Erlaubt das Abspielen der Animation mit dem angegebenen Namen (gleicher Name wie bei BetterModel)

### Configs
- _animations.yml_ - Namen der Animationen (wie aus BetterModel)
- _gui.yml_ - GUI Konfiguration für das Animationsmenü
